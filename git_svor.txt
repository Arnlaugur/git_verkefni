Git spurningar og svör
1.1: cd: Breytir um skrá
1.2: cd ~: Fljótari leið til að komast aftur í heima skránna
1.3: ls: Skrifar út allt sem er í skránni
1.4: pwd: Skrifar út í hvaða skrá þú ert
1.5: mkdir: ef skrá með þessu nafni er ekki til þá býr mkdir hana til
2.1: git clone: clonar geymslu í nýja skrá
2.2: git log: Sýnir allar breytingar í geymslu í öfugri röð
2.3: git status: þessi skypunn sér og skrifar út hvort það er einhver munur á núverandi geymslu og skránni sem notandi er að vinna með
2.4: git diff: skypunns sem sýnir muninn á skránni sem notandi er að vinna með og núverandi skránni
